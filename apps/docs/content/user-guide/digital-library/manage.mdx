# Manage Libraries

In this document, you’ll learn how to manage libraries and their details.

## Create a Library

To create a new library:

1. Go to the Libraries page.
2. Click on the **Create new library** button at the top right.

This opens a new form with different sections. Here you have to enter the Title and Pick Product to attach with library.

:::note

A product can only have one library attached to it. A single product cannot be linked to multiple libraries; it can only be associated with one library at a time.

:::

---

## View Library Details

To view a libraries details:

1. Go to the Library page.
2. Click on the library you want to view.

---

## Delete a Library

:::warning

Deleting a library can’t be undone.

:::

To delete a library:

1. Go to the libraries page.
2. Find the library you want to delete in the Libraries table.
3. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon.
4. Click on Delete Library from the dropdown.
5. Confirm deleting the library by clicking the “Yes, delete” button in the pop-up.
