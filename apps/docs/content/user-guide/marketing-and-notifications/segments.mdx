# Manage Segments

In this guide you will learn how to manage SendGrid segments on InBzar.

## Create a Segment

To create a new segment:

1. Go to the SendGrid → Segments page.
2. Click on the **New Segment** button at the top right.

This opens a new form with different sections on SendGrid.

---

## View Segment Details

To view segment details:

1. Go to the SendGrid → Segments page.
2. Click on the segment you want to view.

---

## Edit a Segment

To edit a segment:

1. Go to the SendGrid → Segments page.
2. Click on the segment you want to edit.

This opens a new form with different sections on SendGrid.

---

### Delete a Segment

:::warning

Deleting a segment can’t be undone.

:::

To delete a segment:

1. Go to the SendGrid → Segments page.
2. Find the segment you want to delete in the Segments section.
3. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon.
4. Click on "Delete" from the dropdown.
