# Configure

In this guide you will learn how to integrate SendGrid with InBzar.

## 1. Create Account

Go to [SendGrid Website](https://sendgrid.com) and create free account.

## 2. Go to Settings

Once you logged in then go **API Keys** by clicking on Settings â†’ API Keys from the sidebar menu.

![SendGrid API Key Menu](/img/sendgrid-api-key-menu.png)

## 3. Create API Key

Create API Key by clicking on top right button **Create API Key**. Give a name and click on **Create & View**

![SendGrid Create API Key](/img/sendgrid-create-api-key.png)

Once API Key is created, you have to copy it.

![SendGrid Copy API Key](/img/sendgrid-copy-api-key.png)

## 4. Configure on InBzar

1. Go to [InBzar Admin Dashboard](https://admin.inbzar.com)
2. Go to settings under extensions click on **SendGrid**.
3. Paste your **API Key** here and click on "Save".

![SendGrid Config](/img/sendgrid-config.png)
