# Manage Automation

In this guide you will learn how to manage SendGrid automation on InBzar.

## Overview

SendGrid Automation is a powerful tool that allows businesses to create automated email workflows, enhancing customer engagement and streamlining communication processes. With SendGrid Automation, users can set up a series of triggered emails based on specific user actions or time intervals, such as welcome emails, onboarding sequences, and cart abandonment reminders. This capability enables businesses to deliver timely and relevant content to their audience, improving customer experience and driving conversions. The platform offers a user-friendly interface for designing and managing automation workflows, with drag-and-drop functionality that makes it easy to visualize the customer journey. Additionally, SendGrid Automation supports dynamic content and personalization, ensuring that each email resonates with the recipient's unique preferences and behaviors. Advanced analytics and reporting tools provide insights into the performance of automated campaigns, helping users optimize their strategies for better results. By leveraging SendGrid Automation, businesses can nurture relationships with their customers efficiently and effectively, enhancing engagement and loyalty over time.

## Create a Automation

To create a new automation:

1. Go to the SendGrid → Automation page.
2. Click on the **New Automation** button at the top right.

This opens a new form with different sections on SendGrid.

---

## View Automation Details

To view automation details:

1. Go to the SendGrid → Automation page.
2. Click on the automation you want to view.

---

## Edit a Automation

To edit a automation:

1. Go to the SendGrid → Automation page.
2. Click on the automation you want to edit.

This opens a new form with different sections on SendGrid.

---

### Delete a Automation

:::warning

Deleting a automation can’t be undone.

:::

To delete a automation:

1. Go to the SendGrid → Automation page.
2. Find the automation you want to delete in the Automation section.
3. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon.
4. Click on "Delete" from the dropdown.
