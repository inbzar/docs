# Manage Notifications

In this guide you will learn how to manage Notifications on InBzar.

## Set From email and name

To set from email and name:

1. Go to the SendGrid → Notifications page.
2. Enter **From email** and optionally **From Name**.

:::not

From email should be verified, You can verify email by clicking on top right button "Verify Sender".

:::

## Change Default Template

All notifications use their own default templates, but you can create your own templates and assign to it.

To change default template:

1. Go to the SendGrid → Notifications page.
2. Find the notification you want to change template.
3. Click on Dropdown in front of notification.
4. Choose your template.
