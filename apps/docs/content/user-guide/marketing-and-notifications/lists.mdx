# Manage Lists

In this guide you will learn how to manage SendGrid lists on InBzar.

## Create a List

To create a new list:

1. Go to the SendGrid → Lists page.
2. Click on the **New List** button at the top right.

This opens a new form with different sections on SendGrid.

---

## View List Details

To view list details:

1. Go to the SendGrid → Lists page.
2. Click on the list you want to view.

---

## Edit a List

To edit a list:

1. Go to the SendGrid → Lists page.
2. Click on the list you want to edit.

This opens a new form with different sections on SendGrid.

---

### Delete a List

:::warning

Deleting a list can’t be undone.

:::

To delete a list:

1. Go to the SendGrid → Lists page.
2. Find the list you want to delete in the Lists section.
3. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon.
4. Click on "Delete" from the dropdown.
