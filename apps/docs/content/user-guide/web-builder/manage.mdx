# Manage Web Builder

This guide will teach you how to manage theme pages effectively. You'll learn how to protect these pages so that only logged-in customers can access them. Additionally, you'll discover how to attach and detach blocks, create new blocks, or use existing ones to customize your pages according to your needs.

## View Page Details

To view a pageâ€™s details:

1. Go to Web Builder.
2. Click on the page you want to view its details

This opens the details page.

---

## Create a Page

To create a new Page:

1. Go to Web Builder.
2. Click on **Create Theme Page** at top right button.
3. This will open new form, fill required fields.

:::note

If you want pages can only be accessed by logged-in customers. Enable **Is Protected**

:::

---

## Add Blocks to Page

1. Click on the page. (This will open a new section)
2. In new opened section you have two options
    1. Select existing block
        To select existing block click on the block and will attached to page.
    2. Create new block
        1. To create new block click on **Create Block** at left bottom.
        2. In the new section pick block type
        3. Fill the required fields and click on **Save and close**

---

## Change Block Order

To change blocks order in page.

1. Go to page's detail
2. Use drag button to rearrange blocks.

---

## Edit a Page

### Change Page Status

:::info

Changing a page status to published makes it available on store. Changing a page status to draft makes it unavailable on store.

:::

To change a page status:

1. Go to the web builder.
2. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon.
3. Choose from the dropdown the opposite status (for example, Draft)

### Detach Block from Page

To remove block from page:

1. Go to page's details
2. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon in front of block.
3. Click on **Detach Block**

:::note

Detaching a block does not actually delete it; the block remains available for use on other pages.

:::

### Edit a Block

To edit existing block:

1. Go to page details
2. 2. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon in front of block.
3. Click on **Edit**
4. Make changes and click on **save**

:::note

Editing a block will affect all pages where that block is used. Any changes made to the block will be reflected across all instances, ensuring consistency and saving time when updating content that appears on multiple pages.

:::

### Delete a Block

:::warning

Deleting a block can't be undone.

:::

To delete block:

1. Go to page details
2. 2. Click on the <UiIcon lightIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999308/Medusa%20Docs/UI%20Icons/1ordBC6_ssysel.png" darkIcon="https://res.cloudinary.com/dza7lstvk/image/upload/v1667999326/Medusa%20Docs/UI%20Icons/dSwWYBH_stzgoi.png" alt="three dots" /> icon in front of block.
