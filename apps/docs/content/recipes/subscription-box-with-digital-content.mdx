# Subscription Box with Digital Content

In this guide you will learn how to create Subscribe Boxes with digital content attach to it.

## Overview

Subscription boxes are a unique product offering that combine multiple items into a single box, providing a curated experience for customers. Subscription boxes are a rapidly growing sector in e-commerce, offering a convenient and personalized shopping experience. They provide customers with a variety of products delivered regularly to their doorsteps, often based on a specific theme or niche.

## Get Started

### 1. Create a Product

To create a new product:

1. Go to the Products page.
2. Click on the New Product button at the top right.
3. Go to Subscription section and enable "Subscription Box"

![Subscription Box Section](/img/subscription-section.png)

That's it you just have to enable **Subscription Box**, then create product as normal. Learn more about how to create product in [Manage Product user guide](/user-guide/products/manage).

### 2. Create a Library

To create a new library:

1. Go to the Libraries page.
2. Click on the **Create new library** button at the top right.

This opens a new form with different sections. Here you have to enter the Title and Pick Product which we have created in previous step.

Check user guide how to [Manage Library](/user-guide/digital-library/manage) for more information.

### 4. Create Library files

To create a library file:

1. Go to the Libraries page.
2. Find the library that we created in previous step, click on it.
3. In library detail page click on the **Create new File** button at the top right.

This opens a new form with different sections. Here you have to select the file type and enter required data. 

### 5. File Restrictions

There are two types of restrictions.

1. Restrict file to access without purchase.
2. Periodic Restriction allows you to limit access to digital content based on specific time intervals. Customers can only access this content if they purchase the product within the defined period.

To learn more check [user guide for managing library files restrictions](/user-guide/digital-library/manage-files#file-restrictions)

#### Restrict File
Restrict the file so customer can't access it without purchase.

![Library file restriction setting](/img/file-restriction.png)

:::note

To restrict access to a YouTube video, set its privacy setting to 'Unlisted'. This ensures that users cannot find the video through YouTube search.

:::

#### Periodic Restrictions

Navigate to the periodic restriction section and select the desired time period. In the example image, the file is restricted to customers who purchase the subscription box for July 2024. This ensures that only eligible customers within the specified time frame can access the content.

![Library file periodic restriction](/img/file-periodic-restriction.png)
